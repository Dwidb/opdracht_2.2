Wat betekent dit?

h1, p, a, img, table, ul/ol/li

wat is h1?

H1 is een HTML tag die aangeeft dat het een titel betreft.


wat betekent p?

De 'p' is van het Engelse paragraph, wat alinea betekent.


wat betekent a?

Met het A element wordt een anker gedefinieerd. Aan het A element wordt één van de attributen HREF, NAME of ID toegevoegd.


wat is img?

img staat voor image waardoor je een afbeelding kunt toevoegen.


wat betekent table?

Een tabel is een systeem van rijen (horizontaal) en kolommen (verticaal).


wat is ul en li?

Je begint de ongenummerde lijst met de UL tag - de openingstag voor de lijst. Vervolgens gebruik je de tag LI voor ieder lijstobject (List Item) dat je in de lijst wilt plaatsen.


wat is een ol tag

Het OL element definieert het begin en einde van een geordende lijst, waarvan de items voorafgegaan worden door een volgnummer
